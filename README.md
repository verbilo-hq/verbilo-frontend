# Inspire Dental Group — Clinical Sanctuary

Internal staff intranet for Inspire Dental Group.

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

Push to GitHub and connect your repo to Vercel. Vercel will auto-detect Vite and deploy with:

- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`
